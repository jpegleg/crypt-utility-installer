# crypt-utility-installer

Ansible install for cryptographic test VM software collection.

The playbook uses shell commands with apt-get and apt and is built for Ubuntu 22.


Installs:

<h2>cryptography stuff:</h2>
- gnupg2</br>
- age</br>
- openssl</br>
<h2>pki things:</h2>
- smallstep</br>
- hashicorp vault</br>
- certbot</br>
<h2>compilers</h2>
- go</br>
- cargo (rust)</br>
- g++ (c++)</br>
- gcc</br>
<h2>java stuff</h2>
- openjdk11</br>
- wycheproof</br>
<h2>testing</h2>
- bazel
<h2>cloud client and testing things</h2>
- docker and docker-compose</br>
- kubectl</br>
- aws cli</br>
- gcp cli</br>
- azure cli</br>
